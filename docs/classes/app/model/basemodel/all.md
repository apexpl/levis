
# BaseModel::all 

### Usage

> ModelIterator BaseModel::all([ string $sort_by = 'id' ], [ string $sort_dir = 'asc' ], [ int $limit = '0' ], [ int $offset = '0' ])

### Description

> Get all records from the database table assigned to the model class.

### Parameters

Parameter | Required | Type | Description
------------- |------------- |------------- |------------- 
sort_by | No | string |
sort_dir | No | string |
limit | No | int |
offset | No | int |

### Return
> ModelIterator
